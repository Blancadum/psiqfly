# psiqfly
PsiQFly® — Plataforma de entrenamiento en razonamiento clínico para psicólogos noveles. Recurso didáctico interactivo que entrena el               razonamiento diagnóstico, la mitigación de sesgos          cognitivos y el autocuidado profesional en psicólogos en formación. Desarrollado como TFM en el ISEP (2026).
